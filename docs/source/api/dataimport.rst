=====================================
Additional functions, for data import
=====================================

The module DataImport, contains functions to upload your polarization data in the correct format, ready to be used in the PolCurveFit class. Currently it contains the options to upload data from the Nova software (Metrohm autolab) and the Gamry software. A third functions allows to read column from any other generic textile. Make sure the columns are separated with a space or tab.
